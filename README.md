# Ismail Charouq - Personal Portfolio Website

A professional portfolio website for Ismail Charouq, a TESOL-Certified English Teacher.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Dark Mode** - Toggle between light and dark themes
- **Smooth Animations** - Scroll reveal animations and smooth transitions
- **Interactive Testimonials** - Click on testimonial images to zoom and view in full size
- **Modern UI/UX** - Clean, professional design with gold and navy color scheme

## Sections

- **Hero** - Introduction with call-to-action buttons
- **About** - Personal bio and teaching philosophy
- **Education** - Timeline of educational background
- **Skills** - Teaching and language skills
- **Certificates** - Professional certifications
- **Testimonials** - Feedback from students and colleagues (clickable images)
- **Contact** - Contact information and social links

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript
- Google Fonts (Playfair Display & Inter)

## Usage

Simply open `index.html` in any modern web browser to view the portfolio.

```bash
# On Linux/Mac
xdg-open index.html

# On Windows
start index.html
```

## Customization

### Colors
Edit the CSS custom properties in the `:root` section to change the color scheme:

```css
:root {
    --color-primary: #1a365d;
    --color-accent: #d69e2e;
    /* ... other colors */
}
```

### Content
Update the text content directly in the HTML file to personalize the portfolio.

### Images
Replace the placeholder images with your own:
- `Eaglyy.png` - Hero/profile image
- `1stComment.png` - Testimonial image
- `2ndComment.png` - Testimonial image

## Testimonial Images

The testimonial section features clickable images that can be zoomed for better viewing:
- **Click** on any testimonial image to open it in full-screen mode
- **Press ESC** or **click outside** the image to close
- **Hover** over images to see the "Click to zoom" hint
